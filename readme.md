# Lost (OOP Adventure Game)

![nodemon](https://badgen.net/badge/Nodemon/2.0.15/yellow?icon=git)

Using HTML CSS and JavaScript, I built this mobile-first OOP text game on one page, with functionality to explore rooms, move between rooms, interact with characters, with win/lose conditions. I enjoyed learning object orientated programming in JavaScript. I improved the project by adding a command instructions, objects and icon cues to build a more user friendly interface.
